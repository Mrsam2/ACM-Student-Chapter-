Upcoming Event

Full-Stack Web Development Workshop

Comprehensive workshop covering modern web development technologies including React, Node.js, and database integration

Date March 5, 2025
Time 9:00 AM - 4:00 PM
Venue Computer Lab 1 & 2
Capacity 60 Participants

About the Workshop

Join us for an intensive full-stack web development workshop that will take you from the basics of web development to building complete, modern web applications. This hands-on workshop is designed for both beginners and intermediate developers who want to learn the latest web technologies and best practices.

Over the course of 7 hours, you'll learn to build a complete web application from scratch, covering both frontend and backend development, database integration, and deployment strategies. By the end of this workshop, you'll have the skills to create your own full-stack web applications.

What You'll Learn

Frontend Fundamentals

Master HTML5, CSS3, and JavaScript to create responsive and interactive user interfaces

React.js Development

Build dynamic user interfaces using React components, hooks, and state management

Backend with Node.js

Create robust server-side applications using Node.js and Express.js framework

Database Integration

Learn to work with MongoDB for data storage and retrieval in your applications

Workshop Curriculum

9:00 - 10:30 AM

Session 1: Frontend Fundamentals

  • HTML5 semantic elements and structure
  • CSS3 styling, flexbox, and grid layouts
  • JavaScript ES6+ features and DOM manipulation
  • Responsive design principles
10:45 - 12:15 PM

Session 2: React.js Fundamentals

  • Introduction to React and JSX
  • Components, props, and state
  • Event handling and conditional rendering
  • React hooks (useState, useEffect)
1:15 - 2:45 PM

Session 3: Backend Development

  • Node.js runtime and npm packages
  • Express.js server setup and routing
  • Middleware and error handling
  • RESTful API design principles
3:00 - 4:00 PM

Session 4: Database & Integration

  • MongoDB setup and basic operations
  • Connecting React frontend with Node.js backend
  • CRUD operations and data flow
  • Deployment basics and best practices

Prerequisites & Requirements

Technical Requirements

  • Laptop with Windows/Mac/Linux
  • At least 4GB RAM and 10GB free space
  • Stable internet connection
  • Code editor (VS Code recommended)

Knowledge Prerequisites

  • Basic programming knowledge
  • Familiarity with any programming language
  • Understanding of basic computer concepts
  • No prior web development experience required
